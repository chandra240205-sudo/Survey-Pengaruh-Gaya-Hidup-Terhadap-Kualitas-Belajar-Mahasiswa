# Survey-Pengaruh-Gaya-Hidup-Terhadap-Kualitas-Belajar-Mahasiswa
Analisis statistika Kelompok 5 mengenai pengaruh pola tidur, gawai, dan olahraga terhadap kualitas belajar mahasiswa ITSB menggunakan metode Convenience Sampling. Hasil data membuktikan gaya hidup sehat berbanding lurus dengan efisiensi akademik.
